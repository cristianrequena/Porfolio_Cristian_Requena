Proyectos en Grupo

Fairsplit

Fairsplit es una aplicación que permite dividir y gestionar gastos compartidos en grupo de manera sencilla, mostrando quién debe a quién. Es ideal para viajes o eventos, facilitando la organización de los pagos y saldos.

Proyectos Personales

Plataforma de E-learning

Estoy desarrollando una Aplicación de Gestión Familiar Integral que facilita la organización y el bienestar en el hogar, incluyendo herramientas para listas de tareas, seguimiento de hábitos, planificación de comidas, un historial médico centralizado, etc. Utilizo Node.js, Express y SQL para el backend, y Angular para el frontend, asegurando una experiencia de usuario intuitiva. Esta aplicación ofrece una solución completa para mejorar la comunicación y la gestión de responsabilidades dentro de la familia.
